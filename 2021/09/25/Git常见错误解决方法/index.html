Git常见错误解决方法

1.failed to receive handshake ssl/tls connection failed github 【已解决】

在使用github客户端提交代码时，报错`failed to receive handshake ssl/tls connection failed`
解决方法：$ git config --global http.sslBackend "openssl"

2.在第一次使用git传到远程仓库是报错error: failed to push some refs to ‘github.com/xxx.git’ 【已解决】

​ 经过查资料发现是因为我们在本地新建库后，与远程仓库的内容不一致导致的。为此在我向远程库推送的时候，要先进行pull，让本地新建的库和远程库进行同步。
